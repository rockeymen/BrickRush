# Brick Rush

Brick Rush is a browser arcade defense game built with Three.js. Aim bouncing balls, break falling numbered bricks, choose upgrades, and protect the red line.

## Play Locally

```bash
python3 -m http.server 8010
```

Then open:

```text
http://localhost:8010/
```

## Files

- `index.html` - game page and UI
- `js/` - gameplay, rendering, audio, config, icons
- `assets/` - Three.js runtime and audio assets
